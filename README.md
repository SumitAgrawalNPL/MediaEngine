# 🎬 LocalMediaEngine

A powerful, lightweight desktop application for media file conversion, codec processing, and YouTube video downloading—all running locally on your Windows machine.

## ✨ Features

- **File Type Conversion** – Convert between various media formats effortlessly
- **Format Changing** – Transform file formats while preserving quality
- **Codec Processing** – Convert audio and video codecs from one format to another
- **YouTube Downloading** – Download videos from YouTube directly to MP3 or MP4 format
- **Fully Local** – No cloud uploads, no data tracking—everything runs on your machine
- **User-Friendly Interface** – Intuitive dashboard for easy operation

## 📋 Requirements

### System Requirements
- **OS**: Windows 10/11 (64-bit only)
- **Architecture**: 64-bit processor
- **Storage**: 170MB for the app and sufficient disk space for media files

⚠️ **Important**: This application requires Windows 64-bit OS. Running on 32-bit systems or other operating systems is not supported and may cause crashes.

## 🚀 Getting Started

1. **Download** the latest `.exe` file from the [releases page](https://github.com/SumitAgrawalNPL/LocalMediaEngine/releases)
2. **Run** the executable on your Windows 64-bit machine
3. **Use** the intuitive dashboard to:
   - Select files for conversion
   - Choose desired output format
   - Configure codec settings
   - Download YouTube videos

## 📚 How to Use

Below are step-by-step instructions with screenshots to help you use LocalMediaEngine.

### Converting Media Files

1. Launch the application.
2. Click on "Document and Media Hub"
3. Select the target folder where you want the converted files to be stored
4. Browse or paste the file in the text box
5. from the dropdown menu select the action
6. Finally, click on "Start Processing"

### Downloading YouTube Videos

1. After setting up the target folder, copy the YouTube video URL from your browser.
2. Paste the URL into the "YouTube URL" field in the app.
3. Choose the output format: **MP4** to save video, **MP3** to extract audio.
4. Click **Download** and wait for the download and conversion to complete.

<img width="100%" alt="LocalMediaEngine" src="https://github.com/user-attachments/assets/0753929c-dc72-494a-868f-eb5f1a6163b8" />



Notes:
- Ensure you have the right to download the content. Respect copyright and the platform's terms of service.
- If a download fails, check your internet connection and try again.

---

### Changing Codecs and Extracting Audio or Video

1. After setting up the target folder, copy the YouTube video URL from your browser.
2. If you want to process codec then choose between H.264, H.265, Apple ProRes or AV1, if you want to extract audio or visuals choose between "Extract Audio" or "Extract Video"
3. Upload your file and click "Start Processing" 
4. Wait till its done
